Given a network of roads, a fleet of cars, and a time limit, what is the set of vehicle paths that maximizes the distance covered within the time limit? This Julia package seeks to answer this problem with a set of path routing algorithms including Dijkstra's, greedy BFS solutions, and lookahead trees.

The package includes full documentation, CI pipelines, and 100% unit test code coverage.
